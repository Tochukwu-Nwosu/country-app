# Country App

## About the project
Country App is a simple web app that provides basic details about a country when you enter the fullname of a country.
Details provided are:
 - Flag
 - Coat of Arms
 - Capital
 - Countinent
 - Subregion
 - Population
 - Currency
 - Language
 - Time Zone 
 - Dial Code
 - TLD
 - Traffic
 - Map

## Technologies used
This project was developed using:
 - HTML
 - Tailwindcss
 - JavaScript
